title: 命名规范
---

## HTML/CSS 文件命名

- [强制] 确保文件命名总是以字母开头，且字母一律小写，以 `“-”` 连接且不带其他标点符号

```html
<!-- HTML -->
page.html page-list.html page-detail.html

<!-- css -->
page.css page-list.css page-detail.css
```

## 组件命名

- [强制] 确保组件命名总是以大写字母开头，长文件使用驼峰命名法

```html
<!-- 组件文件夹 -->
BackToTop / index.js

<!-- 组件文件 -->
Edit.vue
```
